	index.php
	login.php
	login-failed.php
	logout.php
	dashboard.php
	password-forgot.php
	password-change.php
	password-changed-successfully.php
	password-change-failed.php
	sitemap.xml
	sitemap.php
	sitemap/\d+.xml (paginated, with the help of sitemap.xml and URL Rewrite: rewrite.ini)
	about-us.php
	contact-us.php
	permission-denied.php (Accessing admin pages without valid login)
	404.php

These pages are almost commonly applied in all projects.

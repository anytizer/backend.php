All the .php files here are meant for specific reasons.

inc.compress.php
	Compress PHP Output

inc.constants.php
	Validate user defined CONSTANTS. If user is allowed to define CONSTANTS,
	but did not define the entire list, it will complete the missing ones.

inc.register_globals.php
	Simulates register_globals = Off configurations

inc.server.php
	Optionally load/feed custom variables.
	This file is out of svn server.
	You can just create and use it here to get included.

inc.config.php
	Main configuration file as a BootStrap Loader for this framework.
